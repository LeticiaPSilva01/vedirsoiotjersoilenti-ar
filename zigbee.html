ZIGBEE

IEEE 802.15.4 WIRELESS MESH STANDARD

01. DEFINIÇÃO

O Zigbee é um protocolo de rede sem fio de baixo consumo, baixa taxa de dados e curto alcance, baseado no padrão IEEE 802.15.4. Ele foi criado para automação e controle, onde a simplicidade do hardware e a longa duração da bateria são mais importantes do que a velocidade de conexão.

02. TOPOLOGIA EM MALHA

O diferencial do Zigbee é a rede Mesh (Malha). Diferente do Wi-Fi, onde todos se ligam ao roteador central, no Zigbee os dispositivos retransmitem os dados uns para os outros, criando uma rede auto-regenerativa.

COORDENADOR
O cérebro da rede. Só existe um por rede e ele inicia o sistema e armazena chaves de segurança.
ROTEADOR
Dispositivos alimentados pela rede elétrica que estendem o sinal e roteiam mensagens.
END DEVICE
Sensores a bateria que apenas enviam dados. Eles "dormem" para economizar energia.

03. SPEC_SHEET

RECURSO VALOR / TIPO
FREQUÊNCIA 2.4 GHz (Global) / 915 MHz (Americas)
ALCANCE 10m a 100m (pode ser estendido via Mesh)
TAXA DE DADOS Até 250 kbps
NODOS NA REDE Até 65.000 dispositivos

VANTAGENS

  • Rede Robusta: Se um roteador falha, os dados acham outro caminho.
  • Extrema Eficiência: Baterias podem durar anos.
  • Segurança: Criptografia AES de 128 bits nativa.

DESVANTAGENS

  • Baixa Velocidade: Inviável para áudio ou vídeo.
  • Interferência: Sofre com Wi-Fi e Micro-ondas (2.4 GHz).
  • Complexidade: Configuração de rede mesh é mais técnica.

04. CASOS_DE_USO

Smart Lighting

Lâmpadas inteligentes (ex: Philips Hue) que formam uma malha pela casa.

HVAC Control

Controle de temperatura e umidade em edifícios comerciais.

Security

Sensores de porta, janela e movimento em sistemas de alarme.
